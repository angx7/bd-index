Modelos de servicio de computo en la nube

Los modelos de servicio de cómputo en la nube son esencialmente las diferentes formas en que se pueden proporcionar y utilizar los servicios de computación en la nube. Existen tres modelos principales de servicio de cómputo en la nube: Infraestructura como Servicio (IaaS), Plataforma como Servicio (PaaS) y Software como Servicio (SaaS)

Infraestructura como Servicio (IaaS): Este modelo proporciona a los usuarios acceso a recursos de computación como servidores, almacenamiento y redes a través de Internet. Los usuarios pueden comprar estos recursos según la demanda, lo que permite una mayor eficiencia y escalabilidad. En este modelo, la instalación y gestión de la infraestructura se delega al proveedor del servicio.

Plataforma como Servicio (PaaS): Este modelo incluye los servicios IaaS, pero proporciona además todas las aplicaciones middleware, es decir, el sistema operativo, la base de datos, el servidor web, etc. En otras palabras, el cliente alquila la explotación de los servidores y de las herramientas integradas. Este modelo facilita el desarrollo e implementación de aplicaciones, haciendo el proceso más simple, eficiente y competitivo.

Software como Servicio (SaaS): Este es el modelo de servicio más conocido. Engloba los dos anteriores y además el proveedor se ocupa de la instalación, configuración, el funcionamiento y el mantenimiento de la interfaz y de los servicios de computación en la nube. Los usuarios acceden a las aplicaciones de software a través de Internet, generalmente pagando una cuota mensual o anual.

Cada uno de estos modelos se ofrece con el fin de resolver requerimientos diferentes basados en las necesidades específicas de cada empresa o usuario.
Además, las empresas pueden combinar diferentes modelos de servicio en la nube para diferentes propósitos.
Es importante mencionar que la elección del modelo de servicio en la nube adecuado dependerá principalmente de cuántas "capas" puedan o quieran externalizar a su proveedor de servicios al migrar de una solución on-premises a una en la nube.
